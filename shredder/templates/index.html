Instructions:
  1. Select "sharps" or "flats" to set accidentals
  2. Choose a key for the scale
  3. Select a preset scale or a *custom* scale
    • For custom scales, enter a name and scale intervals
    • Intervals should include the root note (0) and be seperated by commas
    • For example, the major scale would be: 0,2,4,5,7,9,11
  4. Select a preset tuning or enter a custom tuning
    • For custom tunings, enter uppercase letters (ex: A, G#, Ab, ...) for each string
    • To change the number of guitar strings, change the tuning
    • Whatever is entered in the Tuned box will be used to make the scale
  5. Choose the mode for displaying the scale
    • note: plot the notes on the fretboard
    • degree: plot the degree of each note in the scale
    • interval: plot the interval from each note to the root note
  6. Click the Shred button to build a scale
Shredder scales is also available as a standalone python package

The shredderscales package can be found on PyPI here: shredderscales

use pip for local installation:

	pip install shredderscales

To report issues or view the source code, please visit shredder-scales on github: shredder-scales github
